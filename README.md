# Odin Recipes

A simple, structured recipe website built as part of The Odin Project curriculum. This project serves as a foundational exercise to demonstrate core web development concepts, clean repository management, and semantic markup execution.

## Core Objectives
* Build a multi-page static website using clean, valid HTML.
* Organize directory structures efficiently for local development and production environments.
* Demonstrate a professional Git and GitHub development workflow via the command-line interface.

## Technical Skills Demonstrated
* **Semantic HTML:** Utilizing correct structural tags (`<h1>`, `<p>`, `<ul>`, `<ol>`, `<a>`, and `<img>`) to ensure content accessibility and proper hierarchy.
* **Internal Navigation:** Implementing absolute and relative linking strategies to navigate fluidly between the main index page and specific recipe files.
* **Asset Management:** Implementing local media assets correctly with precise pathing and necessary descriptive attributes.
* **Version Control:** Maintaining structured development histories with semantic commit practices via the terminal.
