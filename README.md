## 🖤 Project 2: Minimalist Landing Page (Cigarettes After Sex)
*   **Directory:** `/Landing-Page`
*   **Core Focus:** Semantic Layout Architecture & Flexbox Alignment

A visually captivating, monochromatic landing page tribute capturing the ambient, cinematic brand identity of the band Cigarettes After Sex. Built to master two-dimensional web layouts without relying on bulky framework systems.

### Core Implementations:
*   **Layout Grid System:** Utilized explicit percentage dimensions (`width: 50%`) paired with Flexbox mechanics to align a fluid top navigation header and an even 4-column music card grid.
*   **Atmospheric Visual Blending:** Integrated custom grayscale filters and `mix-blend-mode: screen` rules to seamlessly dissolve harsh rectangular image borders directly into a pure black backdrop.
*   **Clean Code Structure:** Eliminated deeply nested selector chains (`header ul li a`) by introducing explicit class targeting (`.nav-link`) for robust maintainability.
